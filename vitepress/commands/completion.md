# xlflow completion

Generate shell completion scripts through Cobra.

## Usage

```bash
xlflow completion [bash|fish|powershell|zsh]
```

## When to use

Use this command to install shell completions for interactive terminal use. It is generated by Cobra and does not require Excel COM.

## Example

```powershell
xlflow completion powershell
```

## Output notes

This command writes shell-specific completion script text. It is primarily a human terminal convenience, not part of the workbook automation loop.
